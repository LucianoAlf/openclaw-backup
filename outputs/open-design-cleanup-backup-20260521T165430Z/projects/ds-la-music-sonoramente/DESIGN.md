# SonoraMente LA — Design System

## Open Design Palette Contract
- **Background:** `#FAF8FF`
- **Surface:** `#E8E0F0`
- **Foreground:** `#3D1A6E`
- **Accent:** `#5B2D8E`

## Brand Identity
- **Name:** SonoraMente LA
- **Tagline:** Onde o Som Cuida da Mente
- **Tone:** Acolhedor, científico, empático, premium. Especialista que abraça.
- **Audience:** Pais e responsáveis de crianças 0-12 anos com TEA ou atrasos de desenvolvimento. Profissionais de saúde.

## Colors

### Primary
- **Roxo Profundo:** `#3D1A6E` — Cor primária. Fundos escuros, títulos.
- **Roxo LA:** `#5B2D8E` — Variação média. Degradês.
- **Roxo Claro:** `#7B4DBE` — Destaques suaves.
- **Lavanda:** `#B39DDB` — Elementos delicados, bordas.
- **Fundo claro:** `#FAF8FF` — Fundo claro principal.
- **Dourado:** `#C9A84C` — Acento premium, badges.

### Neutrals
- **Branco:** `#FFFFFF` — Texto sobre fundo escuro.
- **Cinza suave:** `#E8E0F0` — Fundo alternativo.

### NUNCA USAR
- Pink `#E91451` — é da School
- Cores do catavento (amarelo/verde/azul/vermelho) — é da Kids
- Preto pesado `#0A0A0A` — é da School. SonoraMente usa roxo escuro.

## Typography

### Font Families
1. **Playfair Display** — serifada, display. Elegância e cuidado. Títulos e headlines.
2. **DM Sans** — sans-serif, corpo. Legibilidade e modernidade. Textos e subtítulos.

### Scale
| Element | Font | Weight | Size | Notes |
|---|---|---|---|---|
| Hero title | Playfair Display | Bold 700 | 80-120px | Serifada elegante |
| Secondary title | Playfair Display | SemiBold 600 | 48-64px | |
| Subtitle | DM Sans | Medium 500 | 28-36px | |
| Body | DM Sans | Regular 400 | 18-24px | |
| Badge | DM Sans | Bold 700 | 14-16px | |
| CTA | DM Sans | SemiBold 600 | 16-20px | |

### Typography Rules
- Playfair Display pra títulos — nunca pra corpo
- DM Sans pra corpo — nunca pra títulos de impacto
- NUNCA usar Prompt (é da School) ou Volkswagen/Madelina (é da Kids)
- Tom elegante e premium na tipografia

## Layout

### Canvas
- **Instagram carousel:** 1080×1440px (3:4)
- **Render HD:** 2160×2880px (2x)
- **Safe zones:** 180px topo/base

### Grid
- Espaçamento generoso (mais que School/Kids)
- Bordas arredondadas: 16px padrão
- Logo topo centro
- Composição limpa, sem excesso de elementos

## Signature Elements

### 1. Gradientes Suaves
Degradês em roxo (profundo → lavanda). Transições suaves, nunca agressivas.

### 2. Bordas Arredondadas
16px padrão. Cards, imagens, containers — tudo suave.

### 3. Espaçamento Generoso
Mais respiro que School/Kids. Transmite calma e cuidado.

### 4. Elementos Orgânicos
Formas onduladas suaves. Nada geométrico agressivo.

### 5. Logo
SVG oficial:
- Fundo escuro: `logo-sonoramente-horizontal-white.svg`
- Fundo claro: `logo-sonoramente-horizontal-black.svg`
- Solo: `logo-sonoramente-solo-color.svg`
- NUNCA reconstruir com fonte

## Content Rules
- NUNCA usar a palavra "cura" — usar "desenvolvimento", "progresso", "estímulo"
- NUNCA termos capacitistas: "deficiente", "portador", "especial" como eufemismo
- Usar: "criança atípica", "criança com TEA"
- NUNCA falar COM as crianças — falar SOBRE elas PARA os pais
- NUNCA tom alarmista: "se não agir agora..."
- Embasamento científico sempre que possível
- Tom de confiança + acolhimento

## DON'T
- Linguagem clínica fria
- Promessa terapêutica indevida
- Tom alarmista ou que gere culpa nos pais
- Infantilizar os pais
- Misturar elementos School (pink/rock/halftone) ou Kids (catavento/blobs coloridos)
- Visual agressivo ou contrastante demais
- Fontes fora de Playfair Display e DM Sans
